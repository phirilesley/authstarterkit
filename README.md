# 🔐 Enterprise Auth Starter Kit (.NET)

## 📌 Overview

A production-ready authentication and authorization system built with ASP.NET Core, designed for enterprise applications and SaaS platforms.

This project provides a reusable foundation for secure user management, role-based access control, and permission-based authorization.

---

## 🎯 Purpose

To provide a scalable and secure authentication system that can be reused across multiple enterprise systems such as:

* School Management Systems
* Claims Management Systems
* Help Desk Systems
* HR & Payroll Systems

---

## 🚀 Features

* JWT Authentication
* Refresh Token Support
* ASP.NET Core Identity Integration
* Role-Based Authorization
* Permission-Based Authorization
* User Management
* Role Management
* Permission Assignment
* Audit Logging
* Login History Tracking
* Email Confirmation
* Password Reset

---

## 🏗️ Architecture

Clean Architecture:

* API Layer → Controllers
* Application Layer → Business Logic
* Domain Layer → Core Models
* Persistence Layer → Database
* Security Layer → JWT & Permissions
* Auditing Layer → Logs & Tracking
